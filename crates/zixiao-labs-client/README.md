# Zixiao Labs DevOps Platform Client (Yuxu Palace Client)

This is a high-performance embedded client developed by Zixiao Labs, enabling meetings/development/real-time collaboration/continuous integration/deployment/continuous delivery without switching to a web browser.

## Features

Use our services and high-performance CRDT for real-time collaboration in Zed (something not possible with extended hosts).

Manage deployments or CI/CD.

Perform code reviews.

Use with the Zed Agent or AI agents that support ACP.

Publish packages in the Yuxu Palace npm/PyPI/Cargo registry, with the option to sync to the main registry.

Log in to Zed using your Zixiao account (useful in regions with unstable GitHub or on enterprise intranets).

Connect to a self-hosted Yuxu Palace instance.

Status: Not yet released, under development.
